Netcruzer Download V2.03 - 2014 December 9
Added support for iMod and Expansion boards, Buf fixes, Documentation.

Project: Netcruzer Library
Following changes were made to the "src/netcruzer" folder. Contains project independent code.
  • Added new asynchronous I2C Port 1 module.
  • Added support for expansion boards PT66ECI, PT66EI-10P, PT66EI-24P and PT66DIN6.
  • Added support for many new iMod modules.
  • Added reset debug reset command to nz_usbBuffered.c, allowing device to be reset from "Netcruzer USB Terminal".
  • Changed system tick to use TMR1, and reserved TMR1, TMR2 combination for future possible 32-bit tick implementation.
  • Added utick functions for getting 16 and 32-bit 1 us and 8 us tick counts.
  • Many other new functions, improvements and bug fixes.

Netcruzer Download V1.00 Beta - 2012 March 19

Netcruzer Shared Code
Large parts of the Netcruzer Library consists of code from the "Microchip Application Library" (MAL). Some modifications have been made to the MAL. New Netcruzer specific code is located in the "modtronix" folder, and project folders.
  • Added support for Winbond Flash chips used in Netcruzer boards
  • Added secure Firmware upgrade functionality. When firmware is upgraded, new firmware is copied to the external Flash chip. On next power up, the firmware of the SBC is updated with this new firmware.
  • Added code for upgrading firmware via the Web interface.
  • Added bootloader for upgrading firmware via USB application. To do this, jumper must be set to "F" when applying power to board.
  • Added circular buffer (Serial Ports) code, can be used for sending and receiving serial data to/from serial peripherals of the SBC. Lots of future code will use this function
  • Added secure mechanizm for storing and updating application configuration data in the extern EEPROM. A duplicate copy is made of all data modified, ensuring that a power fail during modifications will not cause configuration data to get currupted.
  • Modified HTTP and other code to support webpage uploading via the wget utility

Project: projects/webserver
Following changes were made to the "projects/webserver" project
  • Placed all webpages in "WebPages" folder. Each webpage has it's own folder in this folder
  • Added "Netbeans" folder in "WebPages" folder, with netbeans projects for each webpage
  • Created Ant scrips for selected webpages. These scripts can be used to compile and upload webpage
  • Uploading of web pages in Ant scrips uses wget utility
