Detailed Description

Circular Buffer Structure, for buffers with maximum size of 65535. Exactly the same as BUFCIR structure, but flagBits.bSizePower2 flag must be set. Buffer size must be a power of 2, for example: 2,4,8,16,32,64..... For fast performance, use buffXxx functions. By casting this structure to a BUFCIR, the bufXxx functions can however also be used (not as fast). Because BUFCIR and BUFCIR_FAST are identical, this allows both to be cast to BUFCIR structures, and slower bufXxx functions to be used.

It is important that maxOffsetPow2 does NOT have same name as in BUFCIR structure. Do not rename it to maxOffset. This will prevent buffXxx Macros from knowing at compile time if a BUFCIR or BUFCIR_FAST structure was passed to it!

This structure can be cast to a BUFCIR structure, seeing that BUFCIR and BUFCIR_FAST are identical (only different flags set). When dealing with a pointer to a BUFCIR object in code, the flagBits.bSizePower2 flag can be checked to determine if it is a BUFCIR_FAST object. If this flag is set, optimized buffXxx functions can be used.
