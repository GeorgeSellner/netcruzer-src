Detailed Description

Microcontroller (UC) Port Config. Is used to configure a port of the CPU Ensure all WORDs and DWORDs are on 16-bit boundaries, must be done seeing that this struct can be located in internal FLASH!

Field Documentation

BYTE type

Hardware type. Must be a type that is possible with current hardware. Is used in user interface for selecting type of each pin. A 32, 64, 128 or 256 bit mask can be created for selecting possible functions for each pin. Each bit represents a type. A bitmask is created for each pin, indicating valid types. After selecting type, each type can be configured = typeConf
