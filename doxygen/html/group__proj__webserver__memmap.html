External Flash

The external Flash memory is used to store firmware, configuration and data tables. The XEEMEM_USER_START_ADR and XEEMEM_USER_END_ADR defines should be used to get the address range available for user data. The memory map for the external FLASH is:

0x000000-0x085FFF
(549 KBytes) 		Used for firmware and configuration data.
0x086000-0x086FFF
(4 KBytes) 		Default User Space that can be used by user application to store non valatile data. The start of the user space is given by the XFLASH_USER_SPACE_ADR define in code. User space is followed by File System, which is located at address given by MPFS_RESERVE_BLOCK define. User can use the space in external FLASH from XFLASH_USER_SPACE_ADR up till (MPFS_RESERVE_BLOCK - 1). To increase user space, increase MPFS_RESERVE_BLOCK value.
0x087000-0x3FFFFF
(3,641 KBytes) 		MPFS 2 File System, is used for storing web pages. In code this address range is defined by the MPFS_RESERVE_BLOCK and SPI_FLASH_SIZE defines.
