Detailed Description

Circular Buffer Structure, for buffers with maximum size of 65535. Exactly the same as BUFCIR_FAST structure, but flagBits.bSizePower2 flag must be clear. Use bufXxx functions with this structure. bufXxx functions always work, and buffXxx are optimzed functions for when buffer size is a power of 2.

Examples:
usb/device_MIDI_LCD/main.c.
