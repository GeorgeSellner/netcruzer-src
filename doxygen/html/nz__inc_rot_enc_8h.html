Description

Configuration

The following defines are used to configure this module, and should be placed in the projdefs.h (or similar) file. To configure the module, the required defines should be uncommented, and the rest commented out.

// *********************************************************************
// --- Incremental Rotary Encoder Configuration (from nz_incRotEnc.h) --
// *********************************************************************
//Circular Buffer size
#define INC_ROT_ENC_BUFSIZE 128

Usage

To use this module, the following must be done:

  • Include nz_incRotEnc.h in the c file it is used in.
  • Add nz_incRotEnc.c to the MPLAB project, this is the main driver file.
  • The following additional files are required by nz_incRotEnc.c, and must be added to the project: nz_netcruzer.c
  • In projdefs.h, do any nz_incRotEnc configuration required. This is done by copying the configuration section from the nz_incRotEnc.h files to projdefs.h.
  • In code, call incrot_task() repetitively in main loop
  • All DONE! Can now use this driver's functions!
