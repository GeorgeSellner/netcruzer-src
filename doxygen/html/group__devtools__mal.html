Microchip Application Library
Development Tools

The Microchip Application Library (MAL) is a collection of free software libraries made available by Microchip. They contain code for 8, 16 and 32 bit Microchip PIC processors. MAL is used by Netcruzer projects for TCP/IP, USB and Wireless functionality. It can be downloaded from www.microchip.com/MAL.
We recommend also downloading the "Microchip Application Libraries Help Files" located at the bottom of the page given above. This file contains help documents for all software included with the MAL. Especially useful is the TCP/IP, USB Framework, USB HID Class and WiFi help files.
