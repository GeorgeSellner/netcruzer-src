Description

Functions for reading the DHT11 and DHT22 temperature and humidity values. The DHT_SENSOR_STARTUP_READ_DELAY value will define the initial delay before reading the first values, and is 5 seconds by default.

The DHT_SENSOR_AUTO_READ_PERIOD value will define how long to delay between reading the sensor. The default is 5000ms, meaning the temperature and humidity will automatically be read every 5 seconds. If this value is set to 0, automatic read are disabled. In this case, the

Configuration

The following defines are used to configure this module, and should be placed in the projdefs.h (or similar) file. To configure the module, the required defines should be uncommented, and the rest commented out.

// *********************************************************************
// ------------ DHT22 Configuration (from nz_dhtSensor.h) -------------
// *********************************************************************
#define DHT_SENSOR_TIMEOUT ( 100 ) // Timeout 100ms
#define DHT_SENSOR_STARTUP_READ_DELAY ( 5000 ) // Initial read delay 2 seconds (2000ms)
#define DHT_SENSOR_AUTO_READ_PERIOD ( 5000 ) // Automatically read every 5 seconds (5000ms), set to 0 to disable auto read!
#define DHT_SENSOR_PIN_ARRAY_SIZE ( 128 ) // Size of Pin Array read buffer, must be 4,8,16,32,64,128 or 256. Use 256 for optimum speed!
#define DHT_SENSOR_TYPE ( 2 ) // Sensor Type, 1=DHT11, 2=DHT22
#define DHT_SENSOR_NUMBER_OF_SENSORS ( 1 ) // Number of Sensors

Usage

To use this module, the following must be done:

  • Include nz_dhtSensor.h in the c file it is used in.
  • Add nz_dhtSensor.c to the MPLAB project, this is the main DHT22 driver file.
  • The following additional files are required by nz_dhtSensor.c, and must be added to the project: nz_circularBufferPwr2.c, nz_helpers.c, nz_netcruzer.c and nz_serI2C.c
  • Add "HAS_SERPORT_I2C1" to projdefs.h file.
  • In code, initialize the DHT22. Ensure to delay 300ms from power up. For example:
    delay_ms(300);
    ds2482Init();
  • In projdefs.h, do any DHT22, I2C or other configuration required. This is done by copying the configuration section from the *.h files to projdefs.h. Nothing required, defaults will work!
  • In code, call ds2482Task() repetitively in main loop
  • All DONE! Can now use DHT22 functions defined in nz_dhtSensor.h! Some examples: lcdDisplayString(0, "\fHello\nWorld, Again!"); //Write parsed string to LCD display lcdDisplayChar(0, 'x'); //Display 'x' at current cursor position
