Building the Project

The SBC66EC Board and SBC66ECL Board are programmed with the Webserver Firmware at delivery. All source code for this project is available for free, and can be downloaded via the Netcruzer Download. It is located in the "src/projects/webserver" folder of the download, along with MPLAB X IDE project files for editing and building the source code.

For information on using MPLAB X IDE to build the project, and update the firmware, see Building Projects and Updating Firmware.
