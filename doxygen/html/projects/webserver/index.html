Introduction

This documentation is for the Netcruzer Webserver project. To return to the main Netcruzer documentation page Click Here. It is not a users guide for boards that run the Netcruzer Webserver, but contains information on using, configuring and building the Netcruzer Webserver project. The Netcruzer Download contains all source code, documentation, related applications and scrips required to build and modify this project. This project is located in the ".../src/projects/webserver" folder of the download.

This Netcruzer Webserver uses the TCP/IP and USB stack from the Microchip Application Library. It is currently the main firmware for the Netcruzer SBC66EC and SBC66ECL Single Board Computers. It provides a 10/100 Mbit/sec Ethernet, and a USB HID port. It has a web interface for monitoring, controlling and configuring the board. The USB port is mostly used for debugging, and sending commands and requesting information. The TCP/IP stack supports the following protocols:

  • ARP, IP, ICMP, UDP, TCP, DHCP, SNMP, HTTP, FTP, TFTP
  • Socket support for TCP and UDP
  • Secure Sockets Layer (SSL)
  • NetBIOS Name Service
  • DNS Domain Name System
  • Ethernet Device Discovery
