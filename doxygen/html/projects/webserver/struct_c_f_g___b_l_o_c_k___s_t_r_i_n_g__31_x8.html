Detailed Description

String Table. The string table consists out of blocks, each 8 bytes long. Each entry can either be a 'Header', or character. A 'Header' has a value between 0x80 to 0xff, the bottom 7 bits give length of string to follow. Strings are NOT NULL terminated!

  • A string can occupy a single block if it is 7 characters or less (header byte, string). Format will be: [0x80 | size] [0] [1] [2] [3] [4] [5] [6]
  • If it is longer than 7 bytes, multiple blocks are required. Maximum length is 127 characters. [0x80 | size] [0] [1] ...... [size-1]
  • A service routine can occasionally compact the string table if required. Ensure all WORDs and DWORDs are on 16-bit boundaries, must be done seeing that this struct can be located in internal FLASH!
