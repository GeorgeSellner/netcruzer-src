Default Webpages
Webpages

The Webserver has a HTML Server that is used to display web pages written in HTML and JavaScript. The default web pages at delivery are located in the ".../src/projects/webserver/WebPages/Default" folder of the Netcruzer Download. The ".../src/projects/webserver/WebPages" folder also contains some other sample web pages that can be uploaded to the SBC board.
