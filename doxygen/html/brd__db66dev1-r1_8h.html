#define db66dev1_initButtons ( )    {DIR_BUTTON1=1; DIR_BUTTON2=1; DIR_BUTTON3=1; DIR_BUTTON4=1; PULLUP_BUTTON1=1; PULLUP_BUTTON2=1; PULLUP_BUTTON3=1; PULLUP_BUTTON4=1;}

Initialize all 4 Buttons of the DB66DEV1 board. All button ports are configured as inputs, AND all the Pull-up resistors on all button ports are enabled.

Examples:
db66dev1_demo1/main.c, and db66dev1_demo1_debug/main.c.
#define db66dev1_initLeds ( )    {DIR_LED1=0; DIR_LED2=0; DIR_LED3=0; DIR_LED4=0; DIR_LED5=0; DIR_LED6=0; DIR_LED7=0; DIR_LED8=0;}

Initialize all 8 LEDs of the DB66DEV1 board. All LED ports are configured as outputs.

Examples:
db66dev1_demo1/main.c, and db66dev1_demo1_debug/main.c.
#define db66dev1_setLeds (   val)    {LAT_LED1=val&0x01; LAT_LED2=val&0x02; LAT_LED3=val&0x04; LAT_LED4=val&0x08; LAT_LED5=val&0x10; LAT_LED6=val&0x20; LAT_LED7=val&0x40; LAT_LED8=val&0x80;}

Set all 8 LEDs of the DB66DEV1 board to the given values. Each bit in the given byte represents a LED.

Parameters
valGives the values of the 8 LEDs. Bit 0 (LSB) represents LED1. Bit 7 (MSB) represents LED8.
For Example, to set LED 1, 2 and 8 use db66dev1_setLeds(0b10000011);
For Example, to clear all LEDs use db66dev1_setLeds(0);
Examples:
db66dev1_demo1/main.c, and db66dev1_demo1_debug/main.c.
